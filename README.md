# Experimental shooting game for the terminal built with JS.



[Video](https://www.youtube.com/watch?v=_NAReVkPbsI)

## Install
- `git clone https://github.com/ThePaavero/terminal-claustrophobia.git`

- `cd terminal-claustrophobia`

- `mpm i`

- `npm start`

## Controls
### Moving
`w` UP

`s` DOWN

`a` LEFT

`d` RIGHT

### Shooting
`i` UP

`k` DOWN

`j` LEFT

`l` RIGHT
